# DeveloperLanding API

Backend-сервис для формы обратной связи на сайте разработчика.

Проект реализован на ASP.NET Core Web API с использованием принципов Clean Architecture. 
Сервис принимает сообщения от пользователей, выполняет валидацию, сохраняет заявки, отправляет уведомления по электронной почте и использует AI-анализ для классификации обращений.

---

## 🚀 Возможности

- REST API для обработки формы обратной связи
- Валидация входящих данных
- Отправка email-уведомлений владельцу
- Отправка копии сообщения пользователю
- AI-анализ обращения через OpenRouter API
- Определение:
  - тональности сообщения
  - категории обращения
  - приоритета
- Ограничение частоты запросов (Rate Limiting)
- Глобальная обработка исключений
- Логирование запросов и ошибок
- Swagger документация API
